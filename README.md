# tube-save
Tube Save é uma ferramenta web prática e eficiente desenvolvida com Flask e pytubefix para simplificar o download de vídeos do YouTube. Com uma interface limpa e intuitiva, o usuário só precisa colar o link, clicar em um botão e o Tube Save cuida do processamento pesado, garantindo a melhor qualidade disponível de forma rápida e segura.
